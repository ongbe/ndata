Detailed Description

template<typename T, long ndims>
struct ndata::nvector< T, ndims >

Inherits from ndatacontainer, essentially an ndatacontainer based on a std::vector with some convenience constructors.

Basically the same thing as anndatacontainer<std::vector<T>, T, ndims>, but has the nice property of self-adjusting the size of the underlying std::vector when copy constructed
